## Responsibility

- Handles deploy jobs.
- Cluster agent for klovercloudCD.
- Talks with [Event bank](https://github.com/klovercloud-ci-cd/event-bank) via [Api Service](https://github.com/klovercloud-ci-cd/api-service).
